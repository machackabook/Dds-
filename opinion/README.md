# The Quiet Battlefield: Algorithmic Accountability

[The Quiet Battlefield](https://standardgalactic.github.io/antivenom/opinion/quiet_battlefield.pdf)

* [Algorithmic Harm](https://standardgalactic.github.io/antivenom/opinion/algorithmic-harm.html) — *Audio Overview*

This project explores algorithmic governance not as a future technological threat, but as a present structural reality shaped by administrative systems. The focus is on understanding how decision-making power is embedded in institutions, models, and procurement processes—and how accountability can be restored.

## Core Idea

Algorithmic harm is not primarily a failure of intelligence—it is a failure of structure.  
The most consequential systems are not those that act dramatically, but those that quietly determine outcomes long before a human ever appears in the loop.

Understanding and governing these systems requires shifting attention from spectacle to structure, from models to institutions, and from outputs to decision boundaries.
